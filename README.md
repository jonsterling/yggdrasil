# Yggdrasil

A higher-dimensional type theory based on computads

#### Building

Prerequisites:

* OCaml `4.03.0`
* OPAM `1.2.2`
* Node

```sh
git clone https://github.com/freebroccolo/yggdrasil
cd yggdrasil
npm install
```

#### Running

Run the command line interface:

```sh
$ npm start -- help
```

Run the examples:

```sh
$ npm run start:example
```
