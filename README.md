# Yggdrasil

A higher-dimensional type theory based on computads

#### Building

Prerequisites:

* OCaml `4.03.0`
* OPAM `1.2.2`
* Node

```sh
git clone https://github.com/freebroccolo/yggdrasil
cd yggdrasil
npm install
```

#### Running

Note: currently this prints out some syntax

```sh
npm start
```
